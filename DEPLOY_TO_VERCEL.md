## 🔧 Required: Add Environment Variable

After deploying, you MUST add this for file converter to work:

### In Vercel Dashboard:
1. Go to **Settings** → **Environment Variables**
2. Click **"Add New"**
3. Fill in:
   - **Name:** `CLOUDCONVERT_API_KEY`
   - **Value:** `cloudconvert_your_api_key_here`
   - **Environment:** Production, Preview, Development
4. Click **"Save"**
5. Go to **Deployments** → **"Redeploy"**

### Get CloudConvert API Key:
- Go to: https://cloudconvert.com
- Sign up free
- Dashboard → API Keys → Create new
- Copy the key (starts with `cloudconvert_`)

## 🌍 Add Custom Domain (Optional)

To use: `aieditpro.net` instead of vercel.app

### Steps:
1. In Vercel Dashboard → **Settings** → **Domains**
2. **Add Domain:** `aieditpro.net`
3. Add another: `www.aieditpro.net`
4. **Update DNS** at your domain registrar:

```
Type: CNAME
Name: www
Value: cname.vercel-dns.com

Type: CNAME
Name: @
Value: cname.vercel-dns.com
```

5. **Wait 5-60 minutes** for DNS to update
