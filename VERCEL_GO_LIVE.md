## ⚡ QUICK START (5 Minutes)

### Prerequisites (Verify):
```bash
# Make sure code is on GitHub
git status
# Should show: "On branch main" and "working tree clean"

# If not, push:
git add .
git commit -m "Ready for Vercel"
git push origin main
```

### Deploy Steps:

1. **Go to Vercel:** https://vercel.com
2. **Sign in with GitHub** (or sign up)
3. **Click "Add New..." → "Project"**
4. **Select "aieditpro" repository**
5. **Click "Import"**
6. **Configuration appears (all auto-detected):**
   - Framework: Next.js ✓
   - Build: npm run build ✓
   - Output: .next ✓
7. **Add Environment Variables:**
   ```
   EMAIL_SERVICE=gmail
   EMAIL_USER=aieditpronet@gmail.com
   EMAIL_PASSWORD=knia ltjz hrab gnin
   RECIPIENT_EMAIL=aieditpronet@gmail.com
   ```
8. **Click "Deploy"**
9. **Wait 2-3 minutes**
10. **Get live URL (aieditpro.vercel.app)**
11. **Add custom domain (aieditpro.net)**
12. **Update DNS at registrar**
13. **Done!** 🎉
